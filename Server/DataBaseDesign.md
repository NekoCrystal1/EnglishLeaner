# 数据库设计

## 数据库所需模块

### 用户模块

 - 预留 支持使用qq/微信直接登录，手机号登录 的空间
 - 实现支持邮箱登录的功能，并提供接入发送邮箱功能的文档
 - 用户信息中需要包含用户名

### 书籍模块

 - 支持各种书籍，可以用于不同模块，如单词书可以用于单词模块
 - 有独立的书籍存储模块
 - 每本书单独存储
 - 用户可以上传书籍，服务端保存书籍id但不保存内容，内容存于客户端中

### 学习模块

#### 单词模块

 - 每个用户有自己的当前单词计划：包括每日学习量，复习量，选择的书籍
 - 记录每个用户的学习情况：如何记录请自行设计并补充到此处

#### 阅读理解模块

 - 记录学习情况
 - 暂时留空

#### 听说模块

 - 记录学习情况
 - 暂时留空
